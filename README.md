# win-script-launcher

It's a GUI for your scripts. Because double-clicking a `.bat` file is apparently too much work now.

## Just run it
```bash
pip install -r requirements.txt
python script_launcher.py
```

## How to use
1. Dump your `.bat` files into `scripts/`.
2. Open the app.
3. Click the button.
4. Go back to your coffee.

## Dev
- `pytest` for the tests you'll probably ignore.
- `python build_exe.py` if you need an `.exe`.

Don't overthink it. It's just a launcher.
